# 💪 Application Pause du Champion

Application mobile-first permettant de gérer et optimiser ses temps de repos en musculation grâce à des minuteurs personnalisés, un suivi des séances et une interface simple et intuitive, avec une estimation claire et visible du temps restant avant la fin de la séance.

---

## 🚀 Fonctionnalités

- ⏱️ Minuteurs personnalisés pour chaque exercice  
- 🏋️ Gestion des séances (création, modification, sauvegarde)  
- 📊 Suivi de progression  
- 📱 Interface mobile-first (optimisée pour smartphone)  
- ⏳ Estimation du temps restant pour connaître la fin de la séance  
- 💾 Sauvegarde automatique des données (localStorage)

---

## 📲 Installation (utilisation comme application mobile)

### 📱 Sur iPhone (Safari)

1. Ouvre le site dans Safari  
2. Clique sur le bouton Partager (📤 en bas de l’écran)  
3. Sélectionne "Sur l’écran d’accueil"  
4. Clique sur Ajouter  

👉 L’application apparaîtra comme une vraie app sur ton iPhone

---

### 🤖 Sur Android (Chrome)

1. Ouvre le site dans Chrome  
2. Clique sur les 3 points en haut à droite  
3. Appuie sur "Ajouter à l’écran d’accueil"  
4. Confirme  

👉 L’application sera installée comme une application native
